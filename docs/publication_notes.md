# 공개 저장소 업로드 기준

## 포함

- 분석/전처리/시각화/PDF 생성 코드
- 모델 집계 결과 CSV
- 기존 결과와 지역 정제 후 결과 비교표
- 포스터용 방어형 문구
- 최종 PDF 포스터
- 대시보드 프로토타입

## 제외

- `data/raw/`: 원자료 전체
- `data/processed/*.csv`: 행 단위 처리 데이터
- `docs/references/`: 대회 안내문, 보안서약서, 개인 동의서 등 첨부 자료
- `docs/mdis_layouts/`: MDIS 파일설계서 원본
- `output/models/*.pkl`: 로컬 학습 바이너리 모델

## 통계 표현 원칙

- `CPI_TOTAL`은 OR 1.39, p-value 0.090이므로 10% 유의수준의 정책 신호로 표현한다.
- 5% 기준의 확정적 인과 증명이라고 표현하지 않는다.
- LightGBM AUC는 0.537이므로 고성능 예측 모델이 아니라 타겟팅 보조 대시보드로 표현한다.
